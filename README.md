# 🏥 WellnessHub

WellnessHub is an AI-powered healthcare assistant web application that helps users check symptoms, find nearby doctors, and get basic health guidance in a simple and user-friendly way.

---

## 🚀 Features

- 🩺 Symptom Checker (basic health suggestions)
- 🏥 Doctor & Hospital Finder
- 📊 Health Tips and Wellness Guidance
- 💻 Clean and responsive web interface

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (optional)
- Flask (backend, if connected)
